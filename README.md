# PRIMA Algorithmic Trading Bot

This project is an algorithmic trading bot for PRIMA, utilizing pandas and numpy for data analysis and trading logic.

## Structure
- src/: Source code for the bot
- tests/: Unit and integration tests
- requirements.txt: Python dependencies

## Getting Started
1. Install dependencies: `pip install -r requirements.txt`
2. Run the bot: `python src/main.py`

## Features
- Modular design for strategies, data handling, execution, and logging
- Easily extendable for new strategies and exchanges
